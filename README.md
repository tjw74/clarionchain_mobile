# ClarionChain (Android)

## Install — download the APK

1. Open **Releases**:  
   **https://github.com/tjw74/clarionchain_mobile/releases/latest**

2. Under **Assets**, tap **`ClarionChain.apk`** (that exact name — a normal Android installer).

3. Open the downloaded file on your phone and install. If Android asks, allow install from that app (browser / Files).

That’s it. You do **not** need the “raw” view or to rename anything.

---

*Every push to `main` updates the **ClarionChain — Latest** release with a new build. Older numbered releases are optional snapshots.*
